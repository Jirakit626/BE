Use this username and password to login as root 

Username 
```
root
```
Password
```
MYSQL_ROOT_PASSWORD
## Installation



Run Nest JS
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
